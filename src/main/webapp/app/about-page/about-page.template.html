About this project

This a training project we are working on at IRI's Analytics Centre of Excellence (ACE) for our 6-month long internship.

The goal is to learn about the MVC pattern and RESTful APIs; to this end, we use an SQLite database with mock data and build a service around it.

We use Java 8 and Spring at the back-end and JavaScript, AngularJS and AngularJS Material at the front-end. Various other libraries are used; for an extensive list check the README distributed with the source code.

Finally, you can check the sidenav for a list of available pages, or the card below for a list of API endpoints.
